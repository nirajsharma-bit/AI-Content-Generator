# 🤖 AI Content Generator

A full-stack AI-powered content generation web application built using HTML, CSS, JavaScript, Node.js, Express, and OpenRouter API.

## Features

- AI Content Generation
- Blog Generator
- Email Generator
- Instagram Caption Generator
- LinkedIn Post Generator
- Story Generator
- Product Description Generator
- Copy Content
- Download as TXT
- Word Counter
- Character Counter
- Responsive UI

## Tech Stack

Frontend
- HTML
- CSS
- JavaScript

Backend
- Node.js
- Express.js

AI
- OpenRouter API

Version Control
- Git & GitHub

## Run Locally

```bash
cd server
npm install
npm run dev
```

Open:

```
client/index.html
```
